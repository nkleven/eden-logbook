PRE-DEPLOYMENT REQUIREMENTS
================================================================================

Azure Resources (Must Create First):
   [ ] Azure Subscription with owner access
   [ ] Resource Group
   [ ] Cosmos DB (NoSQL) with databases: eden-inventory
   [ ] Key Vault (Premium tier recommended)
   [ ] Storage Account (product images)
   [ ] Application Insights (optional but recommended)

Third-Party Services:
   [ ] Auth0 Tenant (identity provider)
   [ ] Stripe Account (payment processing)

Secrets to Store in Key Vault:
   - cosmos-key (Cosmos DB primary key)
   - auth0-secret (session encryption, 32+ chars)
   - auth0-client-secret (from Auth0 Dashboard)
   - stripe-secret-key (from Stripe Dashboard)
   - stripe-webhook-secret (from Stripe Dashboard)

Environment Variables:
   - ADMIN_EMAILS (comma-separated email list)
   - AUTH0_BASE_URL (your production domain)
   - COSMOS_ENDPOINT (Cosmos DB endpoint URL)
   - AZURE_KEYVAULT_URL (Key Vault URL)
